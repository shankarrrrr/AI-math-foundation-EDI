# Screenshots

This folder contains screenshots for the README and documentation.

## How to Add Screenshots

1. Take screenshots of each module
2. Save them with descriptive names:
   - `dashboard.png` - Main dashboard
   - `gradient-descent.png` - Gradient descent module
   - `neural-network.png` - Neural network visualizer
   - `pca.png` - PCA module
   - `feature-space.png` - Feature space classifier
   - `convolution.png` - Convolution filters
   - `ml-model.png` - ML model trainer
   - `vectors.png` - Vector visualization
   - `matrices.png` - Matrix operations
   - `transformations.png` - Linear transformations
   - `systems.png` - Systems of equations
   - `eigenvalues.png` - Eigenvalue visualization

3. Optimize images (recommended size: 800px width for main, 400px for side-by-side)

## Screenshot Guidelines

- Use the platform in action (not empty states)
- Show interesting examples
- Capture the full module interface
- Use high-quality PNG format
- Keep file sizes reasonable (< 500KB each)
